HTML : HyperText Markup Language

ウェブとはいくつかのウェブページがハイパーテキストリンクで蜘蛛の巣状に結びついたもの

トップページはindex.htmlと名付けるのが一般的

ブラウザと検索エンジン

ブラウザと検索エンジンは別物。
  • ブラウザの例 : Google Chrome, Safari, Microsoft Edge
  • 検索エンジンの例 : Google, Bing, 百度
※Yahoo Japanのようにトップページにニュースなどの色んな情報を載せてるサイトは検索エンジンではなく、ポータルサイトと呼ばれる。
